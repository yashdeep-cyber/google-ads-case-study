# Google Ads Campaign Case Study

## Overview

This campaign tested Google Display Ads for a Hollywood-themed game using a limited budget.

## Objective

- Drive traffic to the game
- Identify high-performing audience segments
- Evaluate ad engagement with a minimal budget

## Strategy

- Targeted movie enthusiasts aged 18–45
- Used affinity audiences
- Focused on curiosity-driven creatives

## Campaign Results

| Metric | Result |
|----------|----------|
| Impressions | 8,229 |
| Clicks | 801 |
| CTR | 9.73% |
| Avg CPC | ₹0.58 |
| Spend | ₹467 |

## Key Insights

- CTR significantly exceeded industry benchmarks
- Hollywood audience segment performed best
- CPC remained extremely low, indicating efficient targeting

## Recommendations

- Implement conversion tracking
- Retarget users who clicked but did not engage
- Improve landing page experience
- Run A/B tests on creatives
